Steps to push code to github :
1. open terminal in vs code (on your html file) and run `git init`
2. `git add filename` -> `git commit -m "message"`
3. now connect your local project to github : make a public repo on the your github account first
4. now in terminal execute `git remote add origin github-profile-url`
5. push the code using `git branch -M main` -> `git push -u origin main`
